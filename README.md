# Boarding Pass 앱

## 아이폰에서 여는 방법

### 방법 1 — GitHub Pages (권장, 무료, GPS 완전 동작)
1. GitHub 계정 만들기 (github.com)
2. 새 Repository 만들기 (이름 예: `boarding-pass`)
3. `index.html` 파일 업로드
4. Settings → Pages → Source: main branch → Save
5. `https://[내아이디].github.io/boarding-pass` 로 Safari에서 열기
6. 위치 권한 허용 → GPS 완전 동작 ✅
